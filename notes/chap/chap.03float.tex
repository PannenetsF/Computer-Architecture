\section{IEEE 标准}

Float 类型 1 位 符号，8 位阶码，23 位尾数。存在向上向下两种溢出。

对于规格化的值， exp 不全为 0 或 1，此时阶码为 \(E = e - Bias\) ，尾数隐含 1 。

对于非规格化的值，exp 全为 0，此时阶码为 \(E = 1 - Bias\) ，尾数不含隐藏位。

对于特殊值，exp 全为 1 ，若尾数为 0 ，则为无穷，不为 0 为 NaN。
